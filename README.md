# 🎉 Sistema de Gestión de Celebraciones

## 📌 Descripción
Aplicación desarrollada en Java para la gestión de celebraciones, permitiendo registrar, visualizar y ordenar eventos por diferentes criterios.

El sistema utiliza persistencia en archivos locales y aplica algoritmos de ordenamiento para organizar la información.

---

## 🚀 Funcionalidades
- Registro de celebraciones
- Persistencia automática en archivo `.txt`
- Ordenamiento de datos:
  - Insertion Sort (ascendente)
  - Merge Sort (descendente)
- Uso de recursividad en procesamiento de datos
- Visualización en tabla (JTable)

---

## 🛠️ Tecnologías utilizadas
- Java
- Swing
- Estructuras de datos (ArrayList)
- Algoritmos de ordenamiento
- Manejo de archivos (File I/O)

---

## 📂 Estructura del proyecto
- `/src` → lógica y GUI
- Archivo `.txt` generado automáticamente

---

## 💾 Persistencia
El sistema crea automáticamente el archivo de almacenamiento en el directorio del usuario, por lo que no requiere archivos iniciales.

---

## ▶️ Ejecución
1. Abrir en NetBeans o IDE compatible
2. Ejecutar clase principal
3. Usar interfaz para gestionar celebraciones
